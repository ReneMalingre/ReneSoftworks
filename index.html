About me

I'm René Malingré, from Rene Softworks, a full-stack web developer and .Net programmer. I'm passionate about creating beautiful, functional websites and applications that fill needs, delight users and solve problems.

I'm a self-taught developer who has been programming for 40 years. I am an optometrist and I have programmed to solve problems and to find efficiencies in my postgraduate studies, past HR roles, and in my own practices. Major projects used commercially include a forecast-driven staff rostering system used by Australia's largest optometry company, a retail point-of-sale system, a clinical record cards system and integrations with Xero, Cliniko and Campaign Monitor.

I am experienced in VB6, VB.Net, C# and I am currently undertaking a 6-month intensive course on full-stack web development through the University of Adelaide.

Accessibility Refactoring

A web marketing firm, Horiseon, required that their website be refactored to conform to web accessibility standards to benefit their users and to improve the site's ranking on search engines. This project entailed adding and improving semantic tags, alt-text, colour contrast and optimising image asset sizes. The result was that the site scored 100% on Google's Lighthouse accessibility score.

Tech:  HTML, CSS

Link:  Live on GitHub Pages
 

Optical Calculator

Optometrists who fit contact lenses often need to do complex calculations to design rigid contact lenses, and sometimes rely on inaccurate rules-of-thumb. This application provides a simple-to-use web-based application to take the complexity out of this problem, giving optometrists confidence that they are ordering the correct specifications for their patients. Watch this project evolve and expand over time, there are a lot of tools to come.

Tech:  HTML, CSS, Javascript

Link:  Live on GitHub Pages

CSS Snippets

CSS has some powerful features and there are styles and layouts that a developer might wish to reuse for new projects. This web application provides a home for easily retrieving these CSS code snippets.

Xero-POS Integration

Daily sales and payments from a custom point-of-sale program are entered into Xero using RESTful API calls. Accounts payable functions allow paying of supplier invoices using a much more efficient user interface than the Xero web UI.

Tech:  VB.Net, C#, RESTful APIs

Link:  Contact Rene for demo

Clinical Records

Bridging the gap between paper-based clinical records and electronic, this application is used in Optometry clinics to store historic scanned records and electronic documents and images, and generates a MS Word doc for new records, turning it into a PDF document and linking it to the patient database on completion.

Tech:  VB.Net, C#
